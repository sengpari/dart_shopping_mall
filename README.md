# 🛍 Dart 장바구니 프로그램

콘솔 환경에서 실행되는 간단한 쇼핑몰 시뮬레이션 앱입니다.  
상품을 목록에서 확인하고, 장바구니에 담고, 총액을 계산할 수 있습니다.

---

## ✨ 주요 기능

- `1`번: 상품 목록 보기  
- `2`번: 장바구니에 상품 담기  
- `3`번: 장바구니 총 가격 보기  
- `4`번: 종료 (안전 종료 확인 포함)

---

## 🧑‍💻 사용 방법

### ▶ 실행

```bash
dart run main.dart
```

### ▶ 예시 흐름

```
[1] 상품 목록 보기 / [2] 장바구니에 담기 / [3] 장바구니 총 가격 보기 / [4] 종료  
번호를 입력하세요: 1  
→ 상품 목록 출력

번호를 입력하세요: 2  
→ 상품명과 수량 입력 → 장바구니 추가
```
